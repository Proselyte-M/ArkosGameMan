# 变更记录

## 1.5

- 新增：第三阶段统一命名与手动编辑改为“仅内存暂存”，统一由底部“保存”批量写入 XML
- 新增：底部“保存/重置”按钮，支持一次性提交或放弃当前未落盘修改
- 新增：切换系统、切换目录或关闭窗口时，统一弹出“保存/放弃/取消”未保存变更提示
- 优化：批量保存增加并发锁与线程写盘，连续点击保存不会重入
- 优化：批量写入失败时按系统回滚已写入文件，保证跨系统写入的原子一致性
- 测试：补充统一命名异常流程与并发写盘完整性回归用例

## 1.4

- 新增：游戏列表支持双击直接启动模拟器
- 新增：模拟器设置第七列改为独立“其他设置”弹窗，支持视频缩放、视频滤镜、音频延迟、帧同步
- 优化：预览开关关闭时同时隐藏图片与视频预览，并暂停视频播放
- 修复：GBA/GB 在误选 `vbam_libretro.dll` 时自动回退 `mgba_libretro.dll`，避免核心启动崩溃
- 优化：内置 Libretro 启动参数支持核心路径覆盖及显示/音频参数透传

## 1.3

- 新增：后台自动检查 GitHub Release 更新并提示用户升级
- 新增：更新下载窗口，支持进度展示与取消下载
- 新增：下载完成后自动替换本地 exe 并重启程序
- 修复：保存 ROM 根目录时保留其它配置段，避免覆盖更新配置

## v1.0.1

- 增强：游戏列表表头改为可交互列宽，支持拖拽调整
- 修复：主题切换后重复报错与界面错乱问题，动画实例改为复用
- 优化：大列表场景下保存元数据优先单行更新，减少整表刷新

## 2026-03

- 修复：收藏视图删除游戏时可能跨系统串写 `gamelist.xml` 的问题
- 修复：搜索输入触发的信号参数不匹配导致的 `TypeError`
- 优化：保存元数据时无变化跳过写盘，降低 I/O 开销
- 增强：关键业务流程增加日志输出（系统刷新、表格刷新、保存、添加、导入）
- 文档：新增 README、架构说明、使用说明、变更记录与 `.gitignore`
