# 使用说明

## 首次启动

1. 运行 `python app.py`
2. 点击“选择目录”并指定 ArkOS `/roms` 根目录
3. 点击“刷新系统”加载系统列表

## 日常操作

## 查看游戏

- 左侧系统列表选择目标系统
- 使用搜索框按名称、ROM 路径、系统名过滤
- 点击表头可切换排序
- 拖动表头分隔线可调整各列宽度

## 编辑元数据

1. 选中游戏
2. 在元数据面板修改字段
3. 点击“保存元数据”

## 收藏管理

- 点击表格第一列心形图标可切换收藏状态
- 进入“收藏游戏”视图按系统分组查看

## 添加/删除/重命名

- 添加 ROM：点击“添加ROM”，选择本地文件
- 删除游戏：右键菜单或“删除游戏”按钮
- 重命名：点击“重命名ROM”

## 导入媒体

- 点击“游戏封面/缩略图/预览视频”标签导入媒体文件
- 程序会自动复制到对应 `media` 子目录并写入相对路径

## 预览

- 有封面时显示图片预览，无封面时隐藏预览画面
- 双击预览图可打开放大查看窗口
- 有视频路径时显示视频预览，无视频时隐藏视频窗口

## 主题与语言

- 顶部按钮可切换亮色/暗色主题
- 语言下拉支持中文、English、日本語、Русский
- 主题切换后如需更细腻过渡，界面会执行短时淡入动画

## 故障排查

- 查看控制台日志定位失败步骤
- 检查系统目录下是否存在 `gamelist.xml.old` 备份
- 若媒体预览异常，优先确认路径是否存在且可读
